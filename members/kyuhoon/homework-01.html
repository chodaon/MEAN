3. console은 어떤 값을 출력하는가

             
                 var x = 1, y = 2;
                 function foo(){
                     var x = y = 7;
                 }
                 foo();
                 console.log(x + y); // 다음의 값은?
                 8 //답
                 // var x는 지역변수, y는 전역변수.
                 // foo에 있는 var x는 밖에 있는 x와 상관없다.
                 // y만 상관있기 때문에, 1+7 = 8이 됨.
